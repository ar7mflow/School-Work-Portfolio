#### **PART 1: Search & Problem Solving (Weeks 1-4)**

   - Agent architectures (simple reflex, model-based, goal-based, utility-based)

2. **Uninformed Search**
   - Breadth-First Search (BFS)
   - Depth-First Search (DFS)
   - Uniform Cost Search (UCS)
   - Depth-Limited Search

3. **Informed (Heuristic) Search**
   - Greedy Best-First Search
   - A* Search (and its optimality proof)
   - Heuristic functions
   - Admissibility and consistency

4. **Local Search & Optimization**
   - Hill Climbing (and variants)
   - Simulated Annealing
   - Genetic Algorithms
   - Local beam search

5. **Adversarial Search (Game Playing)**
   - Minimax Algorithm
   - Alpha-Beta Pruning
   - Expectimax (games with chance)
   - Monte Carlo Tree Search (MCTS)
   - Evaluation functions

#### **PART 2: Constraint Satisfaction (Weeks 5-6)**

6. **Constraint Satisfaction Problems (CSPs)**
   - CSP formulation (variables, domains, constraints)
   - Backtracking search
   - Constraint propagation
   - Arc consistency (AC-3)
   - Forward checking
   - Variable/value ordering heuristics (MRV, LCV, degree)

#### **PART 3: Knowledge & Reasoning (Weeks 6-8)**

7. **Logical Agents**
   - Propositional Logic
   - Syntax and semantics
   - Inference rules (Modus Ponens, Resolution)
   - Conjunctive Normal Form (CNF)
   - SAT solvers

8. **First-Order Logic**
   - Syntax (predicates, quantifiers, functions)
   - Semantics and interpretations
   - Unification
   - Forward and backward chaining
   - Resolution in FOL

#### **PART 4: Probabilistic Reasoning (Weeks 9-12)**

9. **Uncertainty & Probability**
   - Probability axioms
   - Conditional probability
   - Bayes' Rule
   - Independence and conditional independence

10. **Bayesian Networks**
    - Network structure and semantics
    - Conditional probability tables (CPTs)
    - Inference by enumeration
    - Variable elimination
    - **D-Separation** (critical topic)
    - Explaining away

11. **Probabilistic Inference**
    - Exact inference algorithms
    - Approximate inference (sampling methods)
    - Markov Chain Monte Carlo (MCMC)
    - Particle filtering

12. **Decision Making Under Uncertainty**
    - Utility theory
    - Maximum Expected Utility (MEU)
    - Decision networks (influence diagrams)
    - Value of information

### Quick Reference (Always Available):

**Search Complexities:**
| Algorithm | Time | Space | Complete? | Optimal? |
|-----------|------|-------|-----------|----------|
| BFS | O(b^d) | O(b^d) | Yes* | Yes** |
| DFS | O(b^m) | O(bm) | No | No |
| UCS | O(b^(1+⌊C*/ε⌋)) | O(b^(1+⌊C*/ε⌋)) | Yes* | Yes |
| A* | O(b^d) | O(b^d) | Yes* | Yes*** |

*if b finite, **if unit costs, ***if h admissible

**Bayes' Rule:**
$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

**D-Separation Rules:**
1. Serial: A → B → C (A ⊥ C | B)
2. Diverging: A ← B → C (A ⊥ C | B)
3. Converging: A → B ← C (A ⊥ C | ∅, but A ⊥̸ C | B)
