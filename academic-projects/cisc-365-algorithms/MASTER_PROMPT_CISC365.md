## 🔬 QUICK REFERENCE

### Master Theorem

For $T(n) = aT(n/b) + f(n)$ where $a \geq 1$, $b > 1$:

Let $c_{crit} = \log_b a$

**Case 1:** If $f(n) = O(n^c)$ where $c < c_{crit}$
$$T(n) = \Theta(n^{c_{crit}})$$

**Case 2:** If $f(n) = \Theta(n^{c_{crit}} \log^k n)$
$$T(n) = \Theta(n^{c_{crit}} \log^{k+1} n)$$

**Case 3:** If $f(n) = \Omega(n^c)$ where $c > c_{crit}$ AND regularity condition holds
$$T(n) = \Theta(f(n))$$
